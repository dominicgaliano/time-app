## Description

## Installation

To run this project locally, follow the following commands (npm required):

1. Clone the repo

   ```sh
   git clone https://github.com/dominicgaliano/time-app.git
   cd time-app/
   ```

2. Install NPM packages

   ```sh
   npm i
   ```

3. Run local development server

   ```sh
   npm run dev
   ```
